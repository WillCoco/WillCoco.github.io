### new
1.$ hexo new [layout] <title>

### generate
$ hexo generate
生成静态文件。

选项	描述
-d, --deploy	文件生成后立即部署网站
-w, --watch	监视文件变动

### publish
$ hexo publish [layout] <filename>

### server
$ hexo server
启动服务器。默认情况下，访问网址为： http://localhost:4000/。

选项	描述
-p, --port	重设端口
-s, --static	只使用静态文件
-l, --log	启动日记记录，使用覆盖记录格式

### deploy
$ hexo deploy

### clean
$ hexo clean

### 部署
hexo clean
hexo generate
hexo deploy
